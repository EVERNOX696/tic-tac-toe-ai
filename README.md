A Tic tac toe game , where the CPU is as smart as you.

#WHAT ARE THE CHALLENGES?
Finish the game in 20 seconds.
Win the game,or try to atleast ! :p

#Technical Details 
Languages used:-
1.JavaScript (Node.js , ES6 Modules)
2.HTML5
3.CSS

#About the Algorithm used
